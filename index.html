关于本项目

物理与数学原理

  • Box-Muller 变换：将均匀分布随机数转换为高斯分布， 模拟真实电路中的热噪声（Johnson-Nyquist noise）。
  • RC 低通滤波：使用一阶差分方程模拟电容充放电效应， 信号跳变呈指数曲线而非瞬间阶跃。
  • 施密特触发器：通过滞回区间（0.6V - 1.0V）实现抗抖动， 避免噪声导致的逻辑误判。
  • 亚稳态模拟：当 D 输入在时钟边沿处于未定义区间时， 输出随机坍缩，演示建立/保持时间违例的后果。

技术栈

  • TypeScript — 严格类型检查，提升代码可靠性
  • Vite — 现代化构建工具，极速热更新
  • Sass (SCSS) — 模块化样式，变量与 Mixin 复用
  • Canvas API — 高性能实时波形渲染
  • Catppuccin Macchiato — 优雅的配色主题

开发历程

这个项目见证了我从传统前端开发到现代化工程实践的转变。 最初只是一个简单的 HTML + CSS + JavaScript 演示页面，后来逐步迁移到 Vite + TypeScript + SCSS 技术栈。

在这个过程中，我学会了：模块化架构设计、类型安全编程、 ESLint/Prettier 代码规范、以及如何用 Sass 组织复杂样式。 这不仅是一个仿真项目，更是我前端工程化能力的一次完整实践。
